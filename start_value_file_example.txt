# Start value Files

# User has to take care him self, that the start value set fits to the network initialised.
# One Neuron per line in format "value, weight to first neuron in next column, weight to second neuron in next column..."
# First line is first neuron in first column, second line is second neuron in first column...
# Lines with # in the beginning and empty lines will be ignored.
